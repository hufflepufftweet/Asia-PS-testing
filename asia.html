Pertanyaan Umum

Tidak menemukan jawaban? Tanya di Discord kami!

Ya, 100% gratis! Tidak ada biaya registrasi, tidak ada item berbayar. Semua item termasuk Dragon Wings dan item langka tersedia gratis untuk semua player tanpa pengecualian.
Asia-PS support Android (5.0+), iOS (12+), dan PC via emulator BlueStacks atau LDPlayer. Bisa dimainkan di HP, tablet, laptop, maupun komputer desktop.
Buka Growtopia ‚Üí Settings ‚Üí Server ‚Üí masukkan IP: play.asia-ps.net dan Port: 16999. Kamu juga bisa menggunakan patcher otomatis yang tersedia di Discord kami.
100% aman. Asia-PS adalah server terpisah dari Growtopia resmi. Akun resmi Growtopia kamu tidak akan terpengaruh sama sekali. Akun Asia-PS dibuat terpisah.
Event harian (Daily Quest, Login Reward), event mingguan (PvP Tournament, Build Contest), event bulanan (Grand Tournament berhadiah item legendaris), dan seasonal events (Lebaran, Natal, Tahun Baru).
Server berlokasi di Singapura. Ping rata-rata: Indonesia 20-40ms, Malaysia 15-30ms, Filipina 25-50ms, Thailand 30-60ms. Sangat rendah dan stabil untuk gaming!
Dalam 3 tahun operasi, Asia-PS tidak pernah melakukan wipe data. Backup otomatis setiap 4 jam dan full backup harian di multiple server untuk keamanan data kamu.
Hubungi GM melalui Discord di channel #support atau buat ticket di website. GM aktif 24/7 dan rata-rata merespons dalam 30 menit. Ada channel #bug-report khusus untuk laporan bug.
